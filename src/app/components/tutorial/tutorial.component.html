O objetivo do jogo é adivinhar a palavra correta baseada na imagem.

A cada tentativa, a imagem vai sendo revelada aos poucos.

Você tem 5 tentativas para adivinhar a palavra. Cada vez que você erra, uma parte da imagem é revelada em forma de silhueta. Na quinta tentativa errada, a imagem inteira é revelada.

A palavra deve ser digitada no campo de entrada. A cada tentativa incorreta, a imagem será revelada um pouco.

A cada nova tentativa, mais da imagem será revelada.

Se você acertar a palavra antes de usar todas as tentativas, o jogo acaba e a imagem correta será revelada.

A palavra muda todo dia, então você sempre terá um novo desafio!
